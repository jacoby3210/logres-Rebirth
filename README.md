# Королевства Логреса: Возрождение (Web/Canvas, ES Modules)

Одностраничная 2D-стратегия: **карта-исследование (квадратная сетка)** + **тактические бои (гексы 7×5)** + события/лагерь/«Эхо времени».
Проект запускается **без сборки**: просто статический сервер и браузер.

## Как запустить

### Вариант A — Перейдите по ссылке (самый простой)


### Вариант B — Python 
1) Откройте PowerShell/Terminal в папке проекта (там где `index.html`).
2) Запустите сервер:

```powershell
py -m http.server 8000
```

Если команды `py` нет, попробуйте:

```powershell
python -m http.server 8000
```

3) Откройте в браузере:
- `http://localhost:8000/`

### Вариант C — Node.js
Если установлен Node.js:

```powershell
npx http-server -p 8000
```

## Если «сервер не запускается» / открывается старая версия

### 1) Порт занят
Если в консоли ошибка про занятый порт — выберите другой:

```powershell
py -m http.server 8010
```

### 2) Закрыть старый сервер
Окно с сервером нужно **остановить** (Ctrl+C).  
Если окно закрыто, но порт «висит», найдите процесс:

```powershell
netstat -ano | findstr :8000
```

Дальше завершите по PID:

```powershell
taskkill /PID <PID> /F
```

Строка вида `TIME_WAIT` в `netstat` **не означает запущенный процесс** — это нормальное состояние TCP, оно пройдет само.

### 3) Браузер показывает старую версию (кеш)
- откройте DevTools → Network → включите *Disable cache* и обновите страницу (Ctrl+F5);
- либо откройте игру в режиме инкогнито;
- если нужно сбросить прогресс: откройте DevTools → Application → Local Storage и удалите ключи `logres_save_v1`, `logres_prefs_v1`.

## Управление

### Глобально
- **Help**: `H` или `F1`
- **Settings**: `P`
- Закрыть модалку: `Esc` или клик по фону

### Исследование (Overworld)
- Клик по **соседней** клетке — шаг (тратит 1 MP)
- **Конец хода** — восстановить MP
- **Эхо**: `E`
- **Сохранить**: `S`
- **Конец хода**: `Enter`

### Бой (Battle)
- Выберите клетку для **хода** (если не ходили), затем цель для **атаки/лечения**
- **Пропуск**: `Space` / `Enter`
- **Отступление**: `Esc`

## Структура проекта

- `index.html` — точка входа
- `src/main.js` — инициализация canvas + запуск `Game`
- `src/core/*` — движок (input, statemachine, audio)
- `src/gameplay/*` — правила мира/боя/состояния
- `src/ui/*` — UI состояния и оверлеи
- `src/render/*` — рисование (грид, гексы, «рисованные» иконки/юниты)

Документация:
- `docs/TZ.md` — актуальное ТЗ под текущую реализацию
- `docs/STATE_MACHINES.md` — диаграмма/описание переходов состояний
- `docs/ARCHITECTURE.md` — слои и ответственность модулей
