# Королевства Логреса: Возрождение 

Одностраничная **веб-игра** (Canvas + ES Modules), вдохновлённая Heroes of Might and Magic III.
Запускается без Unity/WebSim и без сборки.

## Быстрый запуск
> Нужен локальный сервер (из-за ES modules).

### Вариант A (Python)
```bash
cd logres-rebirth-stage2
python -m http.server 8000
```
Откройте в браузере: `http://localhost:8000`

> Windows 10: если команда `python` не найдена, попробуйте `py -m http.server 8000`.

### Вариант B (Node)
```bash
cd logres-rebirth-stage2
npx serve .
```

## Управление
- Клик по **соседней клетке** — шаг (1 MP).
- **Конец хода** — восстановить MP.
- В бою (пошаговый): клик по цели для активного отряда, **Space/Enter** — пропуск хода, **Esc** — отступление.
- Глобально: **H / F1** — справка, **P** — настройки.

## Что реализовано (Этап 3)
### Карта (Overworld)
- Генерация карты 18x12: враги, элита, босс, лагеря, сундуки + события (**Алтарь / Руины / Аномалия**).
- Подсветка доступных клеток (соседи) и описание клетки по наведению.
- Счётчики оставшихся встреч/событий.

### Бой (7x5 гексов, строго пошаговый)
- Ходит **активный** отряд (подсвечен), выбор цели — кликом.
- **Целитель** может выбрать союзника как цель лечения (только если есть потери).
- Роли: melee / ranged / support, у мага есть splash.
- Визуальные эффекты: всплывающие числа, лёгкая тряска/вспышка (отключается «Меньше анимаций»).
- Лог боя в HUD.

### Экономика и прогрессия
- Единый ресурс **Золото** + **VE (временная энергия)**.
- Награды за бой (gold/VE), простая прогрессия сложности.
- **Аномалия**: риск даёт награду, но усиливает следующую битву.

### Лагеря и Эхо Времени
- Лагерь союзников: восстановление армии за золото, найм 2 типов юнитов, покупка баффов на следующий бой.
- Эхо Времени: восстановление, золото, призыв «Эхо-героя» на следующий бой.

### UX
- **Справка** и **Настройки** доступны всегда (кнопки + hotkeys).
- Настройки: сложность, звук/громкость, подсказки, «меньше анимаций».
- Звук реализован через WebAudio без внешних ассетов.
- Canvas масштабируется под окно браузера.
