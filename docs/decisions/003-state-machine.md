## 2.0 Карта приключений (map)

### 2.1 Глобальная Карта (`map`)
- шаг доступен только на соседнюю клетку;
- шаг возможен только на соседнюю клетку (1 MP), .
- количество MP ограничено, «Конец хода» восстанавливает MP;
- открывается при старте/продолжении игры;

### 2.2 Бой (`battle`)

  — `battleField` (**поле боя**) 
    - (гексы 7×5);
    - **строго пошаговый**: ход = движение (опционально) + действие (атака/лечение);
    - на ходе юнита подсвечивается радиус движения и потенциальные цели. 
    - возможен ход → затем действие.
    - 
  - `battleScout` (**разведка**) - ui где показывается информация об армии игрока и противника, и доступен список действий:
    - атаковать 
    - атаковать (автобой) 
    - поговорить (переход к диалогу)
    - откупиться 
    - отступить
    
  - `battleReport` (**отчёт**) - доступен список действий:
    - переиграть
    - принять
    - завершить игру

### 2.3 Торговля (`trading`)
  - `tradeCamp` — лагерь союзников (лечение/найм/бафы);
  - `tradeMarket` - рынок для обмена ресурсами;

### 2.4 Диалог (`dialogue`)
- дерево реплик с выбором ответа
- используется из `battleScout` (поговорить) и `event`

### 2.5 События (`events`)
  - `event` — алтарь/руины/аномалия (всплывающее событие + фон-анимация)

## 5.0 Игровой цикл

### 5.1 Игровой цикл

Глобальная Карта -> локальная карта 
                 -> событие (глобал)
                 -> сражение  
                 -> торговля
                 -> конец игры
                 
### 5.2 Переходы

- `launcher` → `map`
- `launcher` → `saveload`
- `launcher` → `exit`
- `saveload` → `map`
- `map` → `launcher`

- `end` → `launcher` (начать заново)

- `map` → `battleScout` 
- `battleScout` → `battleField`
- `battleScout` → `battleReport`
- `battleField` → `battleReport`
- `battleReport` → `map`
- `battleReport` → `end`

- `map` → `event` (алтарь/руины/аномалия)

- `map` → `tradeCamp`
- `map` → `tradeMarket`
